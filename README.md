# GIP Landing Page

Landing page institucional para GIP - Gestão do Investimento Público do Piauí

## Stack

- Next.js
- Mantine v7
- Tailwind CSS v4

## Como rodar

```bash
pnpm install
pnpm dev
```

## Estrutura

- `src/app`: layout, providers e estilos globais
- `src/components/SubComponents`: components das Sections que compõem a Landing Page
- `src/data/gip-content.ts`: conteúdo centralizado da página
- `src/lib/theme.ts`: tokens e tema do Mantine
